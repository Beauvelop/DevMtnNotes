var film = {
star: "Matt Damon",
title: "Bourne",
};
function review() {
document.write("The film " + this.title + " stars " + this.star + ".");
}
--- thus far...'.this' is undefined unless we use the call(); method--

review.call(film);

// The film Bourne stars Matt Damon.

So the object (film) that is passed to the call(); method becomes the 'this' in the function
that the call(); method is invoked on--namely--review().


----------Another example (changing the review function a bit)----------
var film = {
star: "Matt Damon",
title: "Bourne",
};
function review(starNumber) {
document.write("The film " + this.title + " stars " + this.star +
" and received "+ starNumber + " out of 5 stars.");
}

**We've specified that review takes an arg. All we need do is pass a second arg to call();


review.call(film, 4); // 'four' (as a str) or 4 (as a Num) both are okay!

The film Bourne stars Matt Damon and received 4 out of 5 stars.
It is important to note that with the call(); method we MUST specify the args independently---Not so, with the apply(); method. --these can be an arbitrary amount.

The apply(); method
does more-or-less the same.
apply(); invokes a function and allows us to specify what '.this' means inside the function invoked.
apply(); also accepts args.

However, instead of accepting 0~>N arguments,
it accepts a single argument after the first argument which is an array to passed into the function after it is invoked.
This is the only difference.
It is useful when we don't know how many args are to be passed to the function being invoked.


----------APPLY(); method can perform the same action on multiple functions----------
var film = {
star: "Matt Damon",
title: "Bourne",
};
function review(starNumber) {
document.write("The film " + this.title + " stars " + this.star +
" and received "+ starNumber + " out of 5 stars.");
}

function suggest(activity, , args){
activity.apply(, args);
}
suggest(review, film, [2]);
The film Bourne stars Matt Damon and received 2 out of 5 stars.
    Args in the apply(); method:
  1. name of function to be invoked
  2. used as '.this'
  3. args as Array passed to the function

------------------------An example of bind();----------------

var heather = {
name: 'Heather',
major: 'Art History'
}
function schedule(lunchCount, classes) {// LunchCount comes first!! IMPORTANT!
if (!this.total) { this.total = 0;
// if it has a property called total --> if not we give it one and set it to zero
}
document.write("Class load: " + classes);
document.write("Total lunchours " + lunchCount);
this.total += classes += lunchCount || 0;
// --if lunchCount isn't fed into the function we just add zero
return this.name + ' will have spent a total of ' + this.total + ' hours at school';
}
var heathersSchedule = schedule.bind(heather, 1);
document.write(heathersSchedule(4));
document.write(heathersSchedule(1));
document.write(heathersSchedule(3));


Class load: 4
Total lunchours 1
Heather will have spent a total of 5 hours at school

Class load: 1
Total lunchours 1
Heather will have spent a total of 7 hours at school

Class load: 3
Total lunchours 1
Heather will have spent a total of 11 hours at school

Lunch hours is constant, but class hours accumulates. Heather's one hour lunch has been bound to the schedule function

The beauty of the bind(); method is that we can set a different '.this' arg (other than heather) and a different arg for the lunch arg if we wish.
