Lynda Git Essentials 3.5 Logging

    View a certain number of commits
    git log -n -number-

    View commits since a date or until a certain date.
    git log --since=2014-03-12
    git log --until=2014-03-12

    View commits by a certain person
    git log --author="Leroy"

    
Remember: grep searches are cool:
    git log --grep="button"

    This searches COMMIT MESSAGE text.
    


    



Lynda Git Essentials 4 SHA-1

    Git generates a checksum --> a SHA(1)

    Each SHA-1 value contains three meta-things

    parent
    author
    commit message


    
Remember: Git's algorithm gives us a 40-digit number is the SHA or the commit ID.
    

  



Lynda Git Essentials 4.3 HEAD

      HEAD is a pointer
      Always to the tip of the current branch in our repo.

      HEAD points to the last state or place in our repo branch.
      Much like a tape recorder.
      Wherever the HEAD is positioned. THAT is where we will start recording next

      
Remember: Checkout another branch and it will go to the tip of that branch.
      

      



Lynda Git Essentials 5.3 Diff delete moving files

      Delete a file
      git rm -f firstFile.txt
      You will have to commit this delete.

      View changes to a file
      git diff

      View git diff in staging index
      git diff --staged

      When moving or changing names, Git identifies as prev as deleted
      and new named file as new file.
      git diff --staged

      
Remember: Git move and git rename are the same operation

          git mv testFile.txt tstFile.txt
